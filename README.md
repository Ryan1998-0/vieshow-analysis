# vieshow-analysis

自動爬取威秀影城指定電影場次的即時售座率，並記錄至 Excel，搭配 Windows 工作排程器定時執行。

---

## 專案背景

想了解特定電影的售票狀況，但威秀官網沒有提供歷史售座資料。
於是寫了這個腳本，每隔一段時間自動記錄各場次的售座狀況，方便事後分析趨勢。

---

## 功能

- 自動開啟威秀官網，點選今天日期
- 依設定篩選指定電影與版本（如：數位英語版）
- 逐一進入每個場次的座位圖，解析已售座位數與總座位數
- 計算售座率，輸出至 `records.xlsx`
- 搭配 Windows 工作排程器，可設定為定時自動執行

---

## 技術棧

- Python 3
- Selenium（網頁自動化）
- openpyxl（Excel 寫入）

---

## 檔案結構

```
vieshow-analysis/
├── main.py                      # 主流程
├── click_hoppers_seat_icon.py   # 核心爬蟲邏輯（Selenium 操作、資料解析、寫入）
├── config.py                    # 監控電影設定
├── records.xlsx                 # 輸出結果（執行後自動產生）
```

---

## 設定方式

編輯 `config.py`，在 `MOVIES` 清單中新增要監控的電影：

```python
MOVIES = [
    {
        "name": "HOPPERS",        # 網頁上顯示的英文名稱（用於搜尋）
        "title": "狸想世界",       # 記錄到 Excel 的顯示名稱
        "versions": ["數位(英)"],  # 要監控的版本，留空 [] 代表監控全部版本
    },
]
```

---

## 執行方式

### 手動執行

```bash
pip install selenium openpyxl
python main.py
```

### 定時自動執行（Windows 工作排程器）

1. 開啟「工作排程器」
2. 建立基本工作，觸發程序設定為「依排程」
3. 動作設定為執行 `python main.py`，指定完整路徑
4. 勾選「以最高權限執行」
